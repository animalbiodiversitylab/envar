# Alpine region

A spatial dataset (class "sf", "data.frame") containing the borders of
the European Alps region, retrieved from the Alpine Convention website
(https://www.alpconv.org/fr/page-daccueil/). Used for examples and
testing within the envar package.

## Usage

``` r
Alps
```

## Format

An sf object :

- name:

  European Alps

- geometry:

  sfc_MULTIPOLYGON geometry

## Source

Alpine Convention

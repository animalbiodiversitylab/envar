# Package index

## Set overall parameters

Function for creating new download task and process it

- [`par_set()`](https://animalbiodiversitylab.github.io/envar/reference/par_set.md)
  : Initialize the Environmental Variable Retrieval Pipeline

## Download variables from different sources

Functions for downloading variables from multiple sources

- [`chelsa()`](https://animalbiodiversitylab.github.io/envar/reference/chelsa.md)
  : Download CHELSA climate data
- [`worldclim()`](https://animalbiodiversitylab.github.io/envar/reference/worldclim.md)
  : Download and process WorldClim Climate Data (Historical & Future)
- [`topography()`](https://animalbiodiversitylab.github.io/envar/reference/topography.md)
  : Download and process EarthEnv Topography layers
- [`accessibility()`](https://animalbiodiversitylab.github.io/envar/reference/accessibility.md)
  : Download and process Global Accessibility Indicators
- [`spectre()`](https://animalbiodiversitylab.github.io/envar/reference/spectre.md)
  : Download and process SPECTRE environmental threat layers
- [`freshwater()`](https://animalbiodiversitylab.github.io/envar/reference/freshwater.md)
  : Download and process EarthEnv Freshwater Environmental Variables
- [`protection()`](https://animalbiodiversitylab.github.io/envar/reference/protection.md)
  : Download and process WDPA Protected Area layers
- [`melc()`](https://animalbiodiversitylab.github.io/envar/reference/melc.md)
  : Download and process MacroEcological Land Cover (MELC)
- [`earthenvlandcover()`](https://animalbiodiversitylab.github.io/envar/reference/earthenvlandcover.md)
  : Download and process EarthEnv land cover variables
- [`geososlandcover()`](https://animalbiodiversitylab.github.io/envar/reference/geososlandcover.md)
  : Download and process Global Simulation Land Use/Cover (GEOSOS) 1 km
  variables
- [`gcamlandcover()`](https://animalbiodiversitylab.github.io/envar/reference/gcamlandcover.md)
  : Download and process Global Future Land Use/Cover layers (2020-2100)
- [`hybridlandcover()`](https://animalbiodiversitylab.github.io/envar/reference/hybridlandcover.md)
  : Download and process Hybrid Global Annual 1-km IGBP Land Cover Maps
- [`pftlandcover()`](https://animalbiodiversitylab.github.io/envar/reference/pftlandcover.md)
  : Download and process Global PFT Land Cover Projections (SSPs-RCPs)
- [`biooracle()`](https://animalbiodiversitylab.github.io/envar/reference/biooracle.md)
  : Download Bio-ORACLE marine data
- [`heterogeneity()`](https://animalbiodiversitylab.github.io/envar/reference/heterogeneity.md)
  : Download and process EarthEnv habitat heterogeneity layers
- [`aridity()`](https://animalbiodiversitylab.github.io/envar/reference/aridity.md)
  : Download and process Global Aridity Index and Potential
  Evapotranspiration (ET0)
- [`cloudcover()`](https://animalbiodiversitylab.github.io/envar/reference/cloudcover.md)
  : Download and process EarthEnv Global Cloud Cover layers
- [`climatezones()`](https://animalbiodiversitylab.github.io/envar/reference/climatezones.md)
  : Download and process Köppen-Geiger climate classification maps
- [`soil()`](https://animalbiodiversitylab.github.io/envar/reference/soil.md)
  : Download and process Harmonized World Soil Database v2.0
- [`soilclimate()`](https://animalbiodiversitylab.github.io/envar/reference/soilclimate.md)
  : Download and process Soil Bioclimatic and Temperature layers
- [`gdppast()`](https://animalbiodiversitylab.github.io/envar/reference/gdppast.md)
  : Download and process Historical Real GDP and Electricity Consumption
- [`population()`](https://animalbiodiversitylab.github.io/envar/reference/population.md)
  : Download and process Global Population Projections (SSP)
- [`habitat()`](https://animalbiodiversitylab.github.io/envar/reference/habitat.md)
  : Download and process IUCN Habitat Classification layers
